
cronus
======

*cronus* is a Cabal Online Emulator with educational purposes, this project
is not and will not be a full emulator the main objective is to study and
increase knowledge on networks, concurrency, database and other fields in game programming.

All Cabal Online art, images, and lore are copyrighted by [ESTsoft Corp][1].

If you want to help develop the project, send an email to jdtsncomp@gmail.com

Required Librarys
-----------------

[Simple Logging Facade for Java (SLF4J) >= 1.7.2][2]

[Apache Log4j >= 1.2.17][3]

[npersistence >= 0.2A build 7][4]

SGDB Librarys
-------------

On development we are using MySQL as database but you can port it to any other
database you need.

[MySQL Connector/J JDBC >= 5.1.18][5]
